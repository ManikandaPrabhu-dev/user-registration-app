**User Registration System - Full Stack Application**

A complete **full stack user registration system** built with *React, Node.js, Express, and MySQL*. 
This application allows users to register with their details,
stores data securely with password hashing, and provides a clean interface for viewing registered users.

Frontend (React)
- ✅ User-friendly registration form with real-time validation
- ✅ Password strength validation (minimum 6 characters)
- ✅ Confirm password matching
- ✅ Phone number validation (10 digits)
- ✅ Success/error message popups
- ✅ Responsive design (works on mobile, tablet, desktop)
- ✅ Navigation between Register and Users pages

- Backend (Node.js + Express)
- ✅ RESTful API with proper status codes
- ✅ Secure password hashing using **bcrypt**
- ✅ MySQL database integration
- ✅ Email uniqueness validation
- ✅ Error handling for duplicate entries
- ✅ CORS enabled for frontend communication

- Database (MySQL)
- ✅ Automatic table creation on server start
- ✅ Secure password storage (hashed)
- ✅ Timestamp tracking for user registration
